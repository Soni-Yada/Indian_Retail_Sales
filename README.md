# Indian Retail Sales 

This project presents a data analysis and visualization of Indian retail sales using Power BI.
The goal is to transform raw retail data into interactive dashboards that help identify trends, seasonal patterns, and key business insights.

Dataset - 
 
 - Source : Kaggle

 - Key Fields :
     - Date
     
     - Region / City
     
     - Product Category
     
     - Sales Revenue
     
     - Quantity Sold
  
Tools Used -

 - Power BI

 - Power  Query (For data cleaning and transformation)

 - DAX (For calculated measures)

Data Cleaning (Power Query)

 - Handled missing values and duplicates.

 - Standardized date formats.
 
 - Cleaned category names.

 - Created structured data model.

Dashboard Development (Power BI)

 - Designed KPI cards using DAX functions to calculate and display Total Sales and Total Profit for quick evaluation.

 - Visualized top and bottom selling categories using charts for clear comparative analysis.

 - Implemented interactive Filters and slicers for better decision making.

 - Applied consistent color schemes and formatting.
